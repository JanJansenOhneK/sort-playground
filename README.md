# Sort Playground
## What is this?
This is a program where you can look at sorting algorithms doing their job, sound included!
## How can i play this?
### 1. Download Python
Download Python 3.12.0 from the official website for your os.
https://www.python.org/downloads/release/python-3120/
### 2. Install needed Modules (pygame)
Execute this command in your terminal:
```bat
pip install pygame
```
Bash alternative:
```bash
sudo apt install python3-pygame
```
### 3. Execute main.py
1. Open your file system to in the folder where you saved this program
2. Right-Click and choose "Open in Terminal"
3. Put this command in the opened terminal:
```bash
python3 main.py
```
